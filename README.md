## Command

#### SAVE link

```shell
stock save "https://www.google.co.kr" folder-name
stock save "https://www.google.co.kr"       // default
stock save https://www.google.co.kr
stock -s https://www.google.co.kr
```

#### link LISTING

```shell
stock list folder-name
stock list      // default
stock -l
```

#### OPEN link list

```shell
stock open folder-name
stock open      // default
stock -o
```
